Technical preview

Representative ingestion snippet
def build_metadata_for_doc(path, tenant_id=None):
    text = extract_text(path)
    clean = normalize_text(text)
    doc_hash = hashlib.sha256(clean.encode()).hexdigest()
    metadata = {
        "source_path": path,
        "tenant_id": tenant_id,
        "hash": doc_hash,
        "published_date": extract_date(clean) or None
    }
    return clean, metadata
Representative: demonstrates deterministic hashing + metadata capture used for deduplication and reproducible ingestion.
