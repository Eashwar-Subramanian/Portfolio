Production Ingestion Pipeline 
def build_metadata_for_doc(path, tenant_id=None):
    """Deterministic metadata extraction for reproducible ingestion."""
    text = extract_text(path)
    clean = normalize_text(text)
    doc_hash = hashlib.sha256(clean.encode()).hexdigest()
    metadata = {
        "source_path": path,
        "tenant_id": tenant_id,
        "hash": doc_hash,
        "published_date": extract_date(clean) or None
    }
    return clean, metadata
Demonstrates deterministic hashing and comprehensive metadata capture for deduplication, audit trails, and reproducible ingestion workflows in production systems.
